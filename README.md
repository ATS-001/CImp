# Cimp - C Programming Guide

[![Next.js](https://img.shields.io/badge/Next.js-15-black?style=flat-square&logo=nextdotjs)](https://nextjs.org/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-v4-38BDF8?style=flat-square&logo=tailwindcss)](https://tailwindcss.com/)
[![React Markdown](https://img.shields.io/badge/React_Markdown-Latest-000000?style=flat-square)](https://github.com/remarkjs/react-markdown)

A modern, interactive study guide built to help students master **Programming in C**, specifically designed for the **KTU (APJ Abdul Kalam Technological University) 2024 Scheme**. The platform combines detailed explanations, code examples, and syllabus-oriented content into an easy-to-use learning experience.

# 📦 Prerequisites

Before running the project, install:

- Node.js **v18 or later**
- npm (comes with Node.js)

Verify installation:

```bash
node -v
npm -v
```

---

# 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/ATS-001/Cimp.git
```

### 2. Navigate into the project

```bash
cd Cimp
```

### 3. Install dependencies

```bash
npm install
```

or

```bash
yarn install
```

or

```bash
pnpm install
```

---

# ▶️ Running the Development Server

```bash
npm run dev
```

or

```bash
yarn dev
```

or

```bash
pnpm dev
```

Open your browser and visit:

```
http://localhost:3000
```
