# Cimp - C Programming Guide

Welcome to **Cimp**, a comprehensive and mobile-friendly web application designed to help students master C Programming, specifically tailored for the KTU 2024 syllabus.

## Features

*   **Comprehensive Topics:** Covers all essential topics from basic storage classes to advanced file handling and dynamic memory allocation.
*   **Code Examples:** Includes clear, well-formatted C code examples for every concept.
*   **KTU Syllabus Aligned:** Content is structured to match the requirements of the KTU 2024 Programming in C syllabus.
*   **Mobile Friendly:** Built with responsive design principles to ensure a seamless experience on both desktop and mobile devices.
*   **Clean UI:** Utilizes Tailwind CSS for a modern, clean, and readable interface.

## Topics Covered

1.  Storage Classes (auto, register, static, extern)
2.  Structure & Union (Differences and examples)
3.  Memory Allocation (malloc, calloc, realloc, free)
4.  File Handling (Modes and functions)
5.  Matrix Problems (Addition, Subtraction, Multiplication, Transpose, Diagonal)
6.  Enumerated Datatypes
7.  Tokens, Keywords, Identifiers
8.  Find Output Problems (KTU Pattern)
9.  Number Programs (Prime, Armstrong, Palindrome, Strong)
10. String Programs (Reverse, String operations)
11. Recursion Programs (Factorial, Sum of digits, Reverse)
12. Sum of N Given Numbers
13. Entry and Exit Controlled Loops
14. Sorting & Searching (Bubble Sort, Linear Search)
15. Structure Programs (Employee, Student, Book details)
16. Functions & Function Prototypes
17. File Related Programs (KTU Pattern)

## Technologies Used

*   **Next.js 15 (App Router):** React framework for building the user interface and handling routing.
*   **Tailwind CSS:** Utility-first CSS framework for styling.
*   **React Markdown:** For rendering the educational content written in Markdown format.
*   **Lucide React:** For beautiful, consistent icons.

## Getting Started

To run this project locally:

1.  Clone the repository.
2.  Install dependencies using \`npm install\`.
3.  Run the development server using \`npm run dev\`.
4.  Open \`http://localhost:3000\` in your browser.
